HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The meta element defines metadata such as the character set, description, keyboards, author and viewport.
  • The script elment embeds client side scripts in the HTML document.
  • The src attribute defines the location or source of the image file.
  • The att attribute contains a text string that describes the appearance and functionality of the image.

    Git

    • git status:checks what branch we are currently on
    • git checkout -b branc-name: creates a new beanch and switches to it
    • The feature/starter-code is the name of the new branch.
    • The -A flag indicates that we want to add all changes.
    • A pull request, or PR is used to merge comparison branch (in our case, the feature branch) into the base branch.
